如果你已经安装好 TouchAI，但还不能真正开始对话，这一页就是用来把模型接起来的：添加服务商、填写 API Key、刷新模型列表，并选好默认模型。

## 支持的服务商

TouchAI 支持接入主流模型服务商，同时你也可以手动填写接口地址和 API Key，接入自定义服务商。

常见选择包括：

- XiaoMIMO
- OpenAI
- Anthropic（Claude）
- Google（Gemini）
- DeepSeek
- 智谱 AI
- Moonshot（Kimi）
- 阿里云通义千问
- xAI（Grok）
- 自定义服务商接口

## 添加服务商

你可以按以下步骤完成配置：

1. 打开 **设置**
2. 进入 **服务商与模型**
3. 点击添加一个新的服务商
4. 填写服务商名称、接口地址和 API Key
5. 保存后点击刷新模型列表
6. 选择一个默认模型

![服务商与模型页面](/settings/model-configuration/model-configuration-overview.png)

如果保存成功，并且模型列表能正常刷新出来，你就可以在 TouchAI 里选中默认模型，并直接发起对话了。

## 模型切换

在日常使用中，你可以随时在对话界面快速切换模型：

1. 点击输入框上方的模型图标
2. 或使用快捷键 `Ctrl+M`
3. 从列表中选择当前要使用的模型

日常使用时，你可以随时在对话界面临时切换模型；这不会影响你在设置里选定的默认模型。

## 安全说明

你的 API Key 仅保存在本地设备中，绝不会上传到 TouchAI 服务器。

在填写密钥时，建议注意以下几点：

- 不要将密钥分享给他人
- 优先保存在自己的账号体系中
- 如果怀疑密钥泄露，请及时到对应服务商平台更换

## 常见问题

### 模型列表刷新不出来？

通常可以优先检查以下几点：

1. 接口地址是否填写正确
2. API Key 是否有效
3. 本地网络连接是否正常
4. 服务商官方接口是否可用
