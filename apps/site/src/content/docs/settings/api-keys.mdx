title: 配置供应商
description: 常用模型提供商的 API Key 获取方式
---

如果你准备配置模型，可以先到对应服务商平台获取 API Key。

## XiaoMIMO

1. 打开 [XiaoMIMO Console](https://platform.xiaomimimo.com/console/balance),登录或注册账号
![XiaoMIMO Console 页面](/settings/api-keys/XiaoMIMO/xiaomimo-console-login.png)

2. 进入 **API Key 管理**
![XiaoMIMO API Key 管理](/settings/api-keys/XiaoMIMO/xiaomimo-api-keys-page.png)

3. 创建新的 API Key
![XiaoMIMO 创建 API Key 对话框](/settings/api-keys/XiaoMIMO/xiaomimo-create-key-dialog.png)

4. 填写 API Key 名称
![XiaoMIMO 填写 API Key 名称](/settings/api-keys/XiaoMIMO/xiaomimo-fill-key-name.png)

5. 复制保存，并填写到 TouchAI 中
![XiaoMIMO 填写 API Key](/settings/api-keys/XiaoMIMO/xiaomimo-fill-key.png)

###  首发福利：TouchAI 携手 Xiaomi MiMo 限时免 Tokens
![XiaoMIMO 首发福利](/settings/api-keys/XiaoMIMO/xiaomimo-first-benefit.png)
- **活动时间**：2026 年 6 月 1 日 - 2026 年 6 月 14 日
- **活动权益**：活动期间，**Xiaomi MiMo** 将为 TouchAI 用户提供全系 **限免 Tokens** 支持。
- **参与方式**：登录 TouchAI 即可直接使用，无需额外操作。

## OpenAI

1. 打开 [OpenAI Platform](https://platform.openai.com)，登录或注册账号

![OpenAI Platform 首页](/settings/api-keys/OpenAI/openai-platform-home.png)

2. 进入 **API Keys** 页面

![OpenAI API Keys 页面](/settings/api-keys/OpenAI/openai-api-keys-page.png)

3. 创建新的 API Key
![OpenAI 创建 API Key 对话框](/settings/api-keys/OpenAI/openai-create-key-dialog.png)

4. 填写 API Key 名称
![OpenAI 填写 API Key 名称](/settings/api-keys/OpenAI/openai-fill-key-name.png)

5. 复制保存，并填写到 TouchAI 中
![OpenAI 填写 API Key](/settings/api-keys/OpenAI/openai-fill-key.png)

## Google Gemini

1. 打开 [Google AI Studio](https://aistudio.google.com)
![Google AI Studio 登录页面](/settings/api-keys/GoogleGemini/google-gemini-login.png)

2. 登录 Google 账号
![Google AI Studio 登录页面](/settings/api-keys/GoogleGemini/google-gemini-login-2.png)

3. 进入 **Get API Key** 页面
![Google AI Studio Get API Key 页面](/settings/api-keys/GoogleGemini/google-gemini-get-api-key.png)

4. 创建新的 API Key
![Google AI Studio 创建 API Key 对话框](/settings/api-keys/GoogleGemini/google-gemini-create-key-dialog.png)

5. 复制保存，并填写到 TouchAI 中
![Google AI Studio 填写 API Key](/settings/api-keys/GoogleGemini/google-gemini-fill-key.png)
