如果你想直观理解 MCP 服务在 TouchAI 里怎么用，可以先看下面 4 个具体案例：**滴答清单、飞书、淘宝、Search1API**。

这页不再按大类泛泛介绍，而是直接用具体服务说明“这是什么软件、能帮用户做什么、接入后可以怎么用”。这样你更容易判断自己应该先接哪一个。

## 1. 滴答清单

滴答清单是一款任务管理软件，可以帮助用户记录待办、安排提醒、整理项目和日程；如果你想把它通过 MCP 接进 TouchAI，可以先在 [mcp.so 搜索 TickTick 相关 MCP 服务](https://mcp.so/search?q=TickTick)。

- 案例：把今天讨论里提到的 5 件事整理成待办，并按优先级排序，再拆成今天要做和本周要做两组。

![滴答清单案例截图](/extensions/mcp-add-service/mcp-find-service.png)

## 2. 飞书

飞书是一款团队协作与沟通软件，可以帮助用户处理群消息、文档、通知、会议纪要和团队同步；如果你想把它通过 MCP 接进 TouchAI，可以先在 [mcp.so 搜索 Feishu 相关 MCP 服务](https://mcp.so/search?q=Feishu)。

- 案例：整理昨天项目群里的重点结论和待办，再生成一段可以直接发到飞书群里的简短总结。

![飞书案例截图](/extensions/mcp-add-service/mcp-add-service-form.png)

## 3. 淘宝

淘宝是一款电商购物平台，可以帮助用户搜索商品、查看详情、比较规格、价格和评价信息；如果你想把它通过 MCP 接进 TouchAI，可以先在 [mcp.so 搜索 Taobao 相关 MCP 服务](https://mcp.so/search?q=Taobao)。

- 案例：帮我找 3 款适合出差用的双肩包，并整理出容量、重量、价格和评价差异。

![淘宝案例截图](/extensions/mcp-add-service/mcp-view-service-tools.png)

## 4. Search1API

Search1API 是一款联网搜索服务，可以帮助用户检索网页、资讯、资料和结构化搜索结果；如果你想把它通过 MCP 接进 TouchAI，可以先在 [mcp.so 搜索 Search1API 相关 MCP 服务](https://mcp.so/search?q=Search1API)。

- 案例：先搜一下最近关于这个行业的公开资料，再帮我整理成 5 点背景摘要。

![Search1API 案例截图](/extensions/mcp-add-service/mcp-use-service.png)

## 怎么选第一批服务

- **你想把想到的事快速落成待办**：先接滴答清单
- **你想整理群消息、发日报和同步团队**：先接飞书
- **你想比较商品和做购买筛选**：先接淘宝
- **你想先搜资料再继续分析**：先接 Search1API

如果你还不确定怎么接入，直接看 [添加服务](/extensions/mcp-add-service/)。
