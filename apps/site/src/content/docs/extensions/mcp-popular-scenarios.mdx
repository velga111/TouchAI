下面是一些推荐的引入的服务，可以大大增强 TouchAI 的使用体验。

## 1. 滴答清单

[滴答清单](https://dida365.com/) 是一款任务管理软件，可以帮助用户记录待办、安排提醒、整理项目和日程。

1. 打开 [滴答清单 MCP 官方接入说明](https://help.dida365.com/articles/7438132116019216384)，查看接入方式和所需参数
2. 在 Touch 里配置

![滴答清单配置](/extensions/mcp-popular-scenarios/mcp-add-service-form.png)

3. 使用 MCP 服务

![滴答清单案例截图](/extensions/mcp-popular-scenarios/mcp-use-service.png)

## 2. 高德地图

[高德地图](https://www.amap.com/) 是一款地图与出行服务软件，可以帮助用户搜索地点、规划路线、查看通勤时间和周边信息。

1. 打开 [高德地图 MCP 官方接入说明](https://developer.amap.com/api/mcp-server/summary)，查看接入方式和所需参数
2. 在 Touch 里配置

![高德地图配置](/extensions/mcp-popular-scenarios/mcp-open-tool-page.png)


3. 使用 MCP 服务

![高德地图案例截图](/extensions/mcp-popular-scenarios/mcp-use-service1.png)

![高德地图案例截图](/extensions/mcp-popular-scenarios/mcp-use-service2.png)

## 3. Search1API

[Search1API](https://www.search1api.com/) 是一款联网搜索服务，可以帮助用户检索网页、资讯、资料和结构化搜索结果。

1. 打开 [Search1API MCP 官方接入说明](https://github.com/fatwang2/search1api-mcp?tab=readme-ov-file#readme)，查看接入方式和所需参数
2. 在 Touch 里配置mcp-view-service-tools

![Search1API 配置](/extensions/mcp-popular-scenarios/mcp-new-service.png)

3. 使用 MCP 服务

![Search1API 案例截图](/extensions/mcp-popular-scenarios/mcp-view-service-logs.png)
