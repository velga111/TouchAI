MCP 可以理解成 TouchAI 连接外部服务的一种方式。服务接入后，会出现在 **MCP 工具** 里，并能在对话中被直接调用。

## TouchAI 支持连接方式

| 传输类型 | 适配场景 | 所需参数  |
| --- | --- | --- |
| **Stdio** | 服务运行在本机，通过命令启动 | **名称**、**传输类型**、**命令**，以及按需填写 **参数**、**工作目录**、**环境变量** |
| **SSE (Streamable HTTP)** | 服务运行在远程，兼容 Streamable HTTP 或 SSE | **名称**、**传输类型**、**URL**，以及按需填写 **请求头** |
| **HTTP** | 服务运行在远程，通过 HTTP POST 调用 | **名称**、**传输类型**、**URL**，以及按需填写 **请求头** |


## 一个简单判断方法

- 文档里给你一条本机启动命令：通常选 **Stdio**
- 文档里给你一个远程地址，并明确写了 **SSE** 或 **Streamable HTTP**：通常选 **SSE (Streamable HTTP)**
- 文档里给你一个远程地址，并明确写了 **HTTP POST**：通常选 **HTTP**
