TouchAI 的一个重要优势，是它适合**键盘优先、低打断**的使用方式。

你不需要频繁切换鼠标和窗口，就可以快速唤起、输入、切换会话、调整模型，把 AI 更自然地融入当前工作流。

## 核心快捷键

### 窗口管理

| 快捷键 | 操作 |
|----------|--------|
| `Alt+Space` | 切换 TouchAI 窗口 |
| `F11` | 切换窗口最大化 |

> **提示**：你可以在软件的“设置”中自定义全局唤起快捷键（默认为 `Alt+Space`）。

### 消息输入与历史

| 快捷键 | 操作 |
|----------|--------|
| `Enter` | 发送消息 / 确认选择历史记录 |
| `Shift+Enter` | 插入新行 |
| `↑` / `↓` | 切换上一条 / 下一条历史记录或输入历史 |

### 主修饰键快捷键（Ctrl/Cmd + 字母）

按住 `Ctrl`（Windows/Linux）或 `Cmd`（macOS），然后按以下字母：

| 快捷键 | 操作 |
|----------|--------|
| `Ctrl+h` | 打开历史会话列表 |
| `Ctrl+l` | 聚焦搜索输入框 |
| `Ctrl+n` | 开启新会话 |
| `Ctrl+m` | 切换模型选择器 |
| `Ctrl+p` | 切换窗口置顶 |
| `Ctrl+.` | 取消当前请求 |
| `Ctrl+Backspace` | 清空输入框内容 |

## 智能取消逻辑（Esc）

`Esc` 不只是一个单纯的“关闭键”，它会根据当前状态做更合适的处理。

在 AI 正在生成、已有输出内容或已经选择模型时，连续按下 `Esc` 会按顺序执行：

1. **第一次按下**：取消当前 AI 请求
2. **第二次按下**：清空输出内容
3. **第三次按下**：取消当前模型选择
4. **第四次按下**：关闭当前会话窗口

这种设计的目的，是让你在键盘操作下也能顺畅完成“停止、清理、退出”这一整套动作。

## 常用快捷键

如果你刚开始使用，建议优先记住这几个：

- `Alt+Space`：快速唤起 TouchAI
- `Enter`：发送消息
- `Shift+Enter`：换行
- `Ctrl+n`：新建会话
- `Ctrl+h`：打开历史会话
- `F11` ：切换窗口最大化 
- `Ctrl+m`：切换模型
- `Ctrl+p`：切换窗口置顶
- `Esc`：快速取消与退出

先熟悉这几个，通常就已经能明显感受到 TouchAI 的键盘优先优势。
