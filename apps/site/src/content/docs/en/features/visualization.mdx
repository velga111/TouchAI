---
title: Visualization
description: Make code, diagrams, and complex results clearer and easier to read
---

When a result contains code, diagrams, flow charts, or other complex information, TouchAI tries to present it in a way that is easier to read and continue working with, instead of dumping a large block of content all at once.

The goal of this capability is not just to make things look nicer, but to help you understand the key points faster and continue the task more easily.

## How it works

You usually do not need to enable this manually. When a task is better expressed through clearer presentation, TouchAI will prefer a more suitable visual format.

### Code display

When an answer includes code, config, or technical snippets, clearer formatting reduces the reading burden.

Inline code in normal text is also separated more clearly, so keywords, commands, and configuration names are easier to spot.

![Code display example](/features/visualization/visualization-code-display.png)

### Diagrams and structure

Some information is awkward to explain with plain text alone, but becomes much easier to understand once it is shown as a structure diagram, flow chart, or relationship map.
