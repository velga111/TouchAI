If you run into problems, need help, or want to discuss a business partnership, you can reach us through the following channels:

## Issue reporting

If you found a bug or need technical support, we recommend these options:

- Open an issue in [GitHub Issues](https://github.com/TouchAI-org/TouchAI/issues) (recommended)
- Send email to the support team: `support@touch-ai.org`

## Official contact channels

### GitHub
- [TouchAI GitHub repository](https://github.com/TouchAI-org/TouchAI)

### Email
- Support: `support@touch-ai.org`
- Security: `security@touch-ai.org`
- Business: `business@touch-ai.org`
