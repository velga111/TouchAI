## 1.0.0

- **2026-06-01**: `1.0.0` officially launched.
- The first stable release is built around the desktop workflow and supports invoking the app quickly with a shortcut to start a conversation inside the current task.
- Supports configuring model providers and a default model to match different capability and cost needs.
- Includes built-in tools for reading files, searching files, fetching web pages, and running commands.
- Supports modifying files and executing commands to keep code, configuration, and local tasks moving.
- Supports connecting more external services through MCP to expand the range of work TouchAI can handle.
- Provides visual capabilities such as code display and structural diagrams to improve readability for complex results.
- Supports switching to a stronger model when task complexity increases, reducing interruption.
