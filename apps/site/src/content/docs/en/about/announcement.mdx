## Official launch

TouchAI is scheduled for full public launch on **2026-06-01**. Starting now, you can visit the official site and experience the complete product.

TouchAI 1.0.0 focuses on the core experience of **using AI directly inside your current workflow**. The launch version already includes the following capabilities:

- **Instant invoke and keyboard-first interaction**: open the window, start a conversation, switch sessions, and change models quickly with shortcuts while minimizing interruptions.
- **Model configuration and free choice**: connect different providers, refresh available models, and choose a default model.
- **Built-in tools**: read local files, search content, fetch web pages, run commands, and keep processing the result.
- **Code and command collaboration**: modify workspace files and run local commands to keep engineering and troubleshooting tasks moving forward.
- **MCP extension services**: connect external services so TouchAI becomes closer to a real workflow assistant instead of only a chat tool.
- **Visual result presentation**: display code, structural diagrams, reasoning, and other complex results more clearly.
- **Continue when tasks become harder**: switch to a stronger model when needed without manually starting over.
