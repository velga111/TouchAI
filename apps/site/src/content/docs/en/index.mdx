## What is TouchAI?

TouchAI is a desktop AI assistant built around one idea: **bring AI into the task you are already doing instead of making you jump into a separate app**.

You can invoke it with a shortcut, ask questions in the middle of your current task, organize files and notes, or keep reasoning from the context you already have without re-explaining everything from scratch.

## Start with these 3 steps

If this is your first time using TouchAI, complete this path once to get a successful first experience:

<CardGrid stagger>
	<Card title="1. Install TouchAI" icon="download">
		Download and install the app, then make sure it launches correctly on your computer. See the [installation guide](/en/getting-started/installation/).
	</Card>
	<Card title="2. Configure a model" icon="key">
		Add a model provider, enter your API key, and choose a default model. See [model configuration](/en/settings/model-configuration/).
	</Card>
	<Card title="3. Start your first chat" icon="bolt">
		Press `Alt+Space` to open TouchAI, send a simple question, and confirm you receive a reply. See [quickstart](/en/getting-started/quickstart/).
	</Card>
</CardGrid>
