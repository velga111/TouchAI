## Download

Download the latest release from the official [GitHub repository](https://github.com/TouchAI-org/TouchAI) to start your AI desktop workflow.

1. Open the [Releases page](https://github.com/TouchAI-org/TouchAI/releases)
2. Pick the latest version and check the **Assets** section
3. Download the package that matches your system:
   - **Windows**: `.exe` or `.msi`
   - **macOS**: `.dmg`
   - **Linux**: `.deb`, `.rpm`, or `.AppImage`
4. After the download finishes, run the installer and follow the prompts
