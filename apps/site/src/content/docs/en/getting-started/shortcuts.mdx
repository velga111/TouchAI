One of TouchAI's major strengths is that it works well in a **keyboard-first, low-interruption** workflow.

You do not need to switch between the mouse and windows all the time. You can invoke TouchAI quickly, type, switch sessions, change models, and fit AI more naturally into what you are already doing.

## Core shortcuts

### Window management

| Shortcut | Action |
|----------|--------|
| `Alt+Space` | Toggle the TouchAI window |
| `F11` | Toggle maximized window |

> **Tip**: You can customize the global invoke shortcut in the app settings. The default is `Alt+Space`.

### Input and history

| Shortcut | Action |
|----------|--------|
| `Enter` | Send a message / confirm a selected history item |
| `Shift+Enter` | Insert a new line |
| `↑` / `↓` | Move to the previous / next history item or input history |

### Primary modifier shortcuts (Ctrl/Cmd + letter)

Hold `Ctrl` on Windows/Linux or `Cmd` on macOS, then press one of the following keys:

| Shortcut | Action |
|----------|--------|
| `Ctrl+h` | Open the session history list |
| `Ctrl+l` | Focus the search input |
| `Ctrl+n` | Start a new session |
| `Ctrl+m` | Toggle the model picker |
| `Ctrl+p` | Toggle window pinning |
| `Ctrl+.` | Cancel the current request |
| `Ctrl+Backspace` | Clear the input box |

## Smart cancel behavior (`Esc`)

`Esc` is not just a simple close key. It performs a more suitable action depending on the current state.

When the AI is generating, there is already output, or a model is selected, pressing `Esc` repeatedly will perform these actions in order:

1. **First press**: cancel the current AI request
2. **Second press**: clear the current output
3. **Third press**: clear the current model selection
4. **Fourth press**: close the current session window

This design lets you complete the whole “stop, clean up, exit” flow smoothly from the keyboard.

## Shortcuts to memorize first

If you are just getting started, remember these first:

- `Alt+Space`: invoke TouchAI quickly
- `Enter`: send a message
- `Shift+Enter`: insert a new line
- `Ctrl+n`: start a new session
- `Ctrl+h`: open session history
- `F11`: toggle maximized window
- `Ctrl+m`: switch models
- `Ctrl+p`: toggle pinning
- `Esc`: cancel or exit quickly

Once these are familiar, you will usually feel the keyboard-first workflow advantage very clearly.
