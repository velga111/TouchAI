Below are some recommended services that can significantly extend the TouchAI experience.

## 1. TickTick

[TickTick](https://ticktick.com/) is a task management app that helps users capture todos, reminders, projects, and schedules.

1. Open the [TickTick MCP official setup guide](https://help.dida365.com/articles/7438132116019216384) and review the connection method and required parameters
2. Configure it in TouchAI

![TickTick configuration](/extensions/mcp-popular-scenarios/mcp-add-service-form.png)

3. Use the MCP service

![TickTick example screenshot](/extensions/mcp-popular-scenarios/mcp-use-service.png)

## 2. AMap

[AMap](https://www.amap.com/) is a map and travel service that helps users search places, plan routes, estimate commute time, and check nearby information.

1. Open the [AMap MCP official documentation](https://developer.amap.com/api/mcp-server/summary) and review the connection method and required parameters
2. Configure it in TouchAI

![AMap configuration](/extensions/mcp-popular-scenarios/mcp-open-tool-page.png)

3. Use the MCP service

![AMap example screenshot](/extensions/mcp-popular-scenarios/mcp-use-service1.png)

![AMap example screenshot](/extensions/mcp-popular-scenarios/mcp-use-service2.png)

## 3. Search1API

[Search1API](https://www.search1api.com/) is a web search service that helps users retrieve pages, news, references, and structured search results.

2. Configure it in TouchAI
