If you are preparing to configure a model, start by getting an API key from the provider platform you plan to use.

## Launch offer: TouchAI x Xiaomi MiMo limited free tokens

![XiaoMIMO launch offer](/settings/api-keys/XiaoMIMO/xiaomimo-first-benefit.png)

- **Campaign period**: 2026-06-01 to 2026-06-14
- **Offer**: During the campaign, **Xiaomi MiMo** provides **free tokens** across the full model lineup for TouchAI users.
- **How to join**: Sign in to TouchAI and start using it directly. No extra action is required.

## XiaoMIMO

1. Open [XiaoMIMO Console](https://platform.xiaomimimo.com/console/balance) and sign in or create an account
![XiaoMIMO Console page](/settings/api-keys/XiaoMIMO/xiaomimo-console-login.png)

2. Go to **API Key Management**
![XiaoMIMO API Key Management](/settings/api-keys/XiaoMIMO/xiaomimo-api-keys-page.png)

3. Create a new API key
![Create a XiaoMIMO API key](/settings/api-keys/XiaoMIMO/xiaomimo-create-key-dialog.png)

4. Enter an API key name
![Fill in a XiaoMIMO API key name](/settings/api-keys/XiaoMIMO/xiaomimo-fill-key-name.png)

5. Copy and save it, then paste it into TouchAI
![Fill in the XiaoMIMO API key](/settings/api-keys/XiaoMIMO/xiaomimo-fill-key.png)

## OpenAI

1. Open [OpenAI Platform](https://platform.openai.com) and sign in or create an account

![OpenAI Platform home](/settings/api-keys/OpenAI/openai-platform-home.png)

2. Open the **API Keys** page

![OpenAI API Keys page](/settings/api-keys/OpenAI/openai-api-keys-page.png)

3. Create a new API key
![Create an OpenAI API key](/settings/api-keys/OpenAI/openai-create-key-dialog.png)

4. Enter an API key name
![Fill in an OpenAI API key name](/settings/api-keys/OpenAI/openai-fill-key-name.png)

5. Copy and save it, then paste it into TouchAI
![Fill in the OpenAI API key](/settings/api-keys/OpenAI/openai-fill-key.png)

## Google Gemini

1. Open the **Projects** page
![Google AI Studio project dashboard](/settings/api-keys/Gemini/gemini-project-home.png)

2. Create a project
![Google AI Studio create project entry](/settings/api-keys/Gemini/gemini-create-project.png)

3. Enter a project name
![Google AI Studio project name form](/settings/api-keys/Gemini/gemini-fill-project-name.png)

4. Open the **API Keys** page
![Google AI Studio API Keys list](/settings/api-keys/Gemini/gemini-api-keys-page.png)

5. Click **Create API key**
![Google AI Studio create API key entry](/settings/api-keys/Gemini/gemini-create-key.png)

6. Enter the key name and choose the project
![Google AI Studio create API key dialog](/settings/api-keys/Gemini/gemini-create-key-dialog.png)

7. Copy the API key and paste it into TouchAI
![Google AI Studio copy API key](/settings/api-keys/Gemini/gemini-copy-api-key.png)
