If TouchAI is installed but you still cannot start a real conversation, this page is where you connect the model: add a provider, enter an API key, refresh the model list, and choose a default model.

## Supported providers

TouchAI supports mainstream model providers, and you can also enter a custom endpoint and API key to connect your own provider.

Common choices include:

- XiaoMIMO
- OpenAI
- Anthropic (Claude)
- Google (Gemini)
- DeepSeek
- Zhipu AI
- Moonshot (Kimi)
- Alibaba Tongyi Qianwen
- xAI (Grok)
- Custom provider endpoints

## Add a provider

You can finish the setup with the following steps:

1. Open **Settings**
2. Go to **Providers & Models**
3. Add a new provider
4. Fill in the provider name, endpoint URL, and API key
5. Save it and refresh the model list
6. Choose a default model

![Providers and models page](/settings/model-configuration/model-configuration-overview.png)

If the save succeeds and the model list refreshes correctly, you can select a default model in TouchAI and start chatting right away.

## Switch models

During daily use, you can switch models quickly from the chat interface at any time:

1. Click the model icon above the input box
2. Or use the shortcut `Ctrl+M`
3. Choose the model you want to use from the list

You can switch models temporarily in the conversation UI without affecting the default model selected in settings.

## Security note

Your API key is stored only on your local device and is never uploaded to TouchAI servers.

When entering a key, it is still a good idea to remember:

- Do not share the key with others
- Prefer saving it inside your own account system
- If you suspect the key has leaked, rotate it on the provider platform as soon as possible

## Common questions

### Why does the model list not refresh?

Check these points first:

1. Whether the endpoint URL is correct
2. Whether the API key is valid
3. Whether the local network connection is working
4. Whether the provider’s official endpoint is available
