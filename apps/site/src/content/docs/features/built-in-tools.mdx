TouchAI 默认提供一组开箱可用的内置工具，帮助你在当前任务里继续查资料、读内容、运行命令和切换模型。它们不需要额外接入外部服务，适合处理大多数本机和通用任务。

## 内置工具能做什么

- 在本机快速找到文件、截图、PDF 或代码目录
- 读取文件、图片、PDF 与网页内容，并继续整理信息
- 执行终端命令、查看输出并继续排查问题
- 在当前模型不够用时切换到升级链中的更强模型
- 以更清晰的方式展示代码、结构图和复杂结果

## 内置工具都有哪些

| 工具 | 作用 |
|------|------|
| `bash` | 执行本地 shell 命令（带审批机制） |
| `read` | 读取本地文件内容 |
| `file_search` | 搜索本地文件 |
| `web_fetch` | 抓取网页内容 |
| `show_widget` | 展示可视化交互 UI |
| `upgrade_model` | 在需要时切换到更强模型 |
| `VisualizeReadMe` | 读取可视化规范，提升展示效果 |
