---
title: 内置工具
description: TouchAI 默认具备的核心工具能力
---

TouchAI 默认就带着一组可直接使用的内置工具，不需要你先接入额外服务。它们主要用于处理本机文件、网页内容、终端命令、结果展示和模型切换这类通用任务。

你可以把它理解为“让 TouchAI 真正能继续做事”的基础能力，而不只是停留在聊天层面。

### 一览表

| 工具 | 作用 |
|------|------|
| 终端命令`bash` | 执行本地命令、查看输出并继续分析 |
| 读取内容`read` | 打开本地文件、图片或 PDF 内容 |
| 文件搜索`file_search` | 在本机快速定位文件与目录 |
| 网页抓取`web_fetch` | 获取网页内容并继续整理信息 |
| ShowWidget `show_widget` | 在聊天中以内联可交互的方式展示可视化结果 |
| 模型升级`upgrade_model` | 在复杂任务中切换到更强模型 |
