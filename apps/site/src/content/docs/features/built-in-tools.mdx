TouchAI 默认带有一组内置工具，用来帮助你处理部分常用操作。

## 内置工具能做什么

- 查找并读取本地资料
- 修改代码和工作区文件
- 执行命令并继续处理结果
- 抓取网页内容并整理信息
- 把复杂结果展示得更清晰
- 在任务变复杂时切换到更强模型

## 已内置的工具
| 工具 | 作用 |
|------|------|
| `bash` | 执行本地 shell 命令（带审批机制） |
| `read` | 读取本地文件内容 |
| `file_search` | 搜索本地文件 |
| `web_fetch` | 抓取网页内容 |
| `show_widget` | 展示可视化交互 UI |
| `upgrade_model` | 在需要时切换到更强模型 |
| `VisualizeReadMe` | 读取可视化规范，提升展示效果 |
