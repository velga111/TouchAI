TouchAI 通过 `bash` 工具，帮助你高效处理本地任务与终端操作。

## 使用

你不需要先想“该调用什么工具”，而是直接用自然语言描述任务即可。

![代码与命令示意](/features/code-and-command/code-and-command-example.png)

### 审批与边界

部分敏感的终端命令执行需要你的确认。

- 执行前先确认命令用途是否清楚
- 删除、覆盖、重置等高风险命令要格外谨慎
- 如果项目环境、本地权限或工作目录不完整，命令可能无法成功执行

![代码与命令边界](/features/code-and-command/code-and-command-boundary.png)

## 常见问题

### 执行失败常见原因

- 当前工作目录不对
- 缺少依赖或运行环境
- 本地权限不足
- 命令本身就有错误
