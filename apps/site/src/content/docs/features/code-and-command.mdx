### 审批与边界

终端命令的执行是否需要你逐次确认，取决于当前的内置工具设置。使用时建议注意：

- 执行前先确认命令用途是否清楚
- 删除、覆盖、重置等高风险命令要格外谨慎
- 如果项目环境、本地权限或工作目录不完整，命令可能无法成功执行

这项能力的重点是“协助你推进任务”，不是无边界地执行任何命令。

## 常见问题

### 执行失败常见原因

- 当前工作目录不对
- 缺少依赖或运行环境
- 本地权限不足
- 命令本身就有错误
