TouchAI 通过 `bash` 工具，帮助你高效处理本地任务与终端操作。

> **注意**：执行命令会经过严格的审批机制。建议在触发执行前，先确认命令的具体用途与安全性，再继续执行。

###  核心功能：执行命令 (`bash`)

该工具用于在本地执行 Shell 命令，并实时返回终端的输出结果。

* **适用场景**：运行项目的构建、测试和脚本，查看 Git 状态和日志，安装项目依赖，以及排查报错信息等。
* **作用**：让 TouchAI 具备操作终端的能力，自动化完成环境配置与任务执行。

###  常见用法与工作流

你可以通过 `bash` 工具，完成以下自动化任务：

* **执行命令 → 查看结果 → 继续处理错误**
* **查看日志 → 定位问题 → 继续排查**
* **安装依赖 → 运行验证 → 再继续调整**
