---
title: 模型自升级
description: 在任务变复杂时切换到更强模型继续处理
---

TouchAI 通过 `upgrade_model` 工具，在任务复杂度超出当前能力时，自动切换到更强的模型继续推进任务，而无需你手动重新选择。

###  常见触发情况

当遇到以下场景时，模型自升级机制通常会被触发：

* **当前模型能力不足**：处理的任务超出了轻量模型的承载范围。
* **任务突然变复杂**：在对话过程中，需求从简单问答演变成了深度分析。
* **需要更强的专业能力**：任务涉及到复杂的逻辑推理、长文本总结或高难度的代码编写。
* **希望减少手动操作**：期望系统能智能判断并自动切换，保持对话的流畅性。

###  常见用法与工作流

你可以将这一机制视为任务处理的“自动变速箱”：

* **任务变复杂 → 自动升级模型 → 继续完成**
* **从轻量问题开始 → 演变成复杂整理 → 继续深入处理**
