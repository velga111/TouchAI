TouchAI 通过 `file_search` 和 `read` 两个工具，帮助你高效处理本地资料。

> **注意**：TouchAI 的文件搜索能力依赖 Windows 平台上的 **Everything** 工具。为了获得更快、更稳定的搜索体验，请确保你的电脑上已安装并运行 Everything。

### ️ 准备工作：安装与配置 Everything

如果你尚未安装 Everything，请按照以下步骤操作：

1. **下载安装**：前往 [Everything 官方网站](https://www.voidtools.com/zh-cn/downloads/) 下载并安装。
2. **建立索引**：安装完成后首次启动，Everything 会自动为本地文件建立索引（如果文件较多，初次建立可能需要几分钟）。
3. **保持运行**：请确保 Everything 处于后台运行状态，然后再回到 TouchAI 使用文件搜索功能。

**使用前的小建议：**
* 如果搜索结果为空，请优先检查 Everything 是否已经启动。
* 如果刚安装好 Everything，由于索引尚未完成，初期的搜索结果可能不完整，请耐心等待片刻。

###  核心功能

#### 文件搜索 (`file_search`)
该工具用于在海量本地文件中快速定位目标。
* **适用场景**：查找最近保存的文档、搜集参考资料、定位截图、PDF、表格或代码文件等。
* **作用**：在大量文件中快速缩小目标范围，找到你需要的文件路径。

#### 读取文件 (`read`)
当 `file_search` 找到目标后，`read` 工具可以继续打开并读取文件的具体内容。
* **支持格式**：文本文件、代码文件、图片、PDF、目录结构等。
* **作用**：让 TouchAI 在找到文件后，能进一步为你整理信息、总结重点或回答关于文件内容的提问。

###  常见用法与工作流

你可以将这两个工具结合起来，完成以下任务：

* **找文件 → 读内容 → 总结重点**
* **找资料 → 提炼信息 → 继续追问**
* **找目录 → 确认范围 → 再读取具体内容**
