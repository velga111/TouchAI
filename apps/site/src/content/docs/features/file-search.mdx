文件搜索适合“先找到，再继续处理”的任务：你先告诉 TouchAI 想找什么，它帮你在本机定位文件；找到之后，还可以继续读取内容、整理重点或回答问题。

## 适合什么情况

- 只记得文件名的一部分，想先把文件找出来
- 想从截图、PDF、表格、代码文件里继续整理信息
- 想先确认目录范围，再决定下一步处理哪份内容
- 想把“找文件”和“读内容”放在同一次对话里完成

## 使用前准备

TouchAI 在 Windows 上的文件搜索依赖 **Everything** 提供快速索引能力。首次使用前，建议先确认：

1. 电脑上已经安装 [Everything](https://www.voidtools.com/zh-cn/downloads/)
2. Everything 已经启动并完成初始索引
3. 需要搜索的文件没有被索引规则排除

如果 Everything 没有运行，或者刚安装还没建好索引，搜索结果可能为空或不完整。

## 你可以怎么用

文件搜索有两种常见入口：

### 方式 1：直接在输入框输入部分文件名

如果你记得文件名的一部分，可以直接在输入框里输入关键词，例如文档名、扩展名或目录名的一部分。TouchAI 会先帮你定位匹配结果，再继续处理后续内容。

这种方式更适合：

- 你已经记得文件名关键词
- 你想快速缩小结果范围
- 你想先定位文件，再决定下一步读取哪一份

![搜索本地文件](/features/file-search/file-search1.png)

### 方式 2：直接问 AI 帮你搜索

如果你不确定准确文件名，也可以直接用自然语言描述需求，让 TouchAI 先帮你搜索。

常见说法例如：

- 帮我找今天下载的 PDF
- 帮我找项目里所有 README 文件
- 帮我找名字里带“合同”的文档
- 帮我找昨天保存过的截图

这种方式更适合：

- 你只记得文件类型、时间范围或大概用途
- 你想把“搜索 + 阅读 + 总结”放在同一次对话里完成
- 你希望用自然语言直接表达任务

![搜索本地文件](/features/file-search/file-search2.png)

### 搜索时尽量补充这些信息

无论你用哪种方式，补充这些信息都能让结果更准确：

- 文件名关键词
- 文件类型（如 PDF、PNG、Markdown）
- 时间范围（如“今天下载的”“昨天改过的”）
- 大概位置（如“项目根目录里”“下载文件夹里”）

### 找到后继续读取和整理

定位到文件后，你可以继续让 TouchAI：

- 打开并总结重点
- 读取图片或 PDF 内容
- 对比多份文件的差异
- 回答关于这份文件的具体问题

## 成功后你会看到什么

搜索成功时，TouchAI 通常会先返回匹配结果或文件路径，然后你可以继续指定要读哪一份内容。  
如果结果为空，通常说明：

- 关键词不够准确
- Everything 尚未运行或索引未完成
- 文件不在当前可搜索范围内

## 常见问题

### 搜不到文件怎么办？

优先检查：

1. Everything 是否已经启动
2. 文件是否刚创建，索引还没来得及更新
3. 关键词是否过于宽泛或过于具体
4. 文件是否位于被排除的目录中

### 结果太多怎么办？

可以补充更多条件，例如文件类型、时间范围、所在目录或更具体的关键词，帮助 TouchAI 缩小范围。

### 找到文件后怎么继续读内容？

你可以直接继续追问，例如“读取第一份 PDF 并总结重点”或“打开这个文件并告诉我哪里有风险”。
