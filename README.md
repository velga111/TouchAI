### 功能特性

- **一触即达** - `Alt+Space` 召之即来挥之即去，不打断工作流
- **全键盘操作** - 支持无鼠标全键盘操作，极大提升效率
- **桌面上下文** - 文件、屏幕、剪贴板智能感知
- **工具支持** - 内置7大工具，支持MCP工具拓展
- **BYOK** - 无厂商锁定，自由切换提供商和模型
- **特色功能** - 文件搜索、可视化交互UI、模型自升级
